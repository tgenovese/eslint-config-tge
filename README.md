# eslint-config-tge

Opinionated ESLint [shareable config](http://eslint.org/docs/developer-guide/shareable-configs.html)


## Install

```bash
$ npm install --save-dev eslint-config-tge
```


## Usage

Add some ESLint config to your `package.json`:

```json
{
    "name": "awesome-project",
    "eslintConfig": {
        "extends": "tge"
    }
}
```

Or to `.eslintrc.js`:

```js
module.exports = {
  extends: [
    'tge'
  ]
}
```
